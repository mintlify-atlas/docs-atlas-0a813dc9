## What is Binance Futures Signal Bot?

Binance Futures Signal Bot is a professional-grade automated trading system built for cryptocurrency futures markets. It connects to signal providers — Telegram channels, TradingView webhooks, or custom APIs — and executes trades on your behalf with configurable leverage, dynamic position sizing, and a trailing stop-loss engine.

The bot is designed for traders who want to automate their strategy execution without building exchange integrations from scratch. It handles order placement, position tracking, risk management, and live P&L reporting through a rich terminal interface, all from a single `python main.py` command.

## Key Features

### Core Engine

- **Multi-exchange support** — Trade on Binance Futures, Bybit USDT Perpetual, or OKX Perpetual Swap from the same configuration file.
- **Telegram signal integration** — Listen to any Telegram channel or bot and parse incoming trade signals automatically.
- **TradingView webhook receiver** — Accept alerts from TradingView strategies and indicators via a local HTTP webhook server on a configurable port.
- **Custom API signal source** — Connect a proprietary signal source through the custom API adapter.
- **Configurable leverage** — Set leverage from 1x up to 125x per exchange limits.
- **Dynamic position sizing** — Allocate a percentage of your account balance per position using `max_position_pct`.
- **Trailing stop-loss engine** — Lock in profits by following the price at a configurable percentage distance once a position is in profit.

### Strategies & Tools

- **EMA Crossover** — Long/short signals on fast (default 9) vs. slow (default 21) EMA crossovers.
- **RSI Divergence** — Detects bullish and bearish price/RSI divergences with adjustable period and thresholds.
- **Bollinger Band Breakout** — Enters positions when price closes outside the upper or lower band.
- **MACD Momentum** — Trades histogram crossovers with signal line confirmation and zero-line trend filter.
- **Real-time P&L dashboard** — Live profit and loss overview rendered in the terminal.
- **Position manager** — View all open positions and close them individually or in bulk.
- **Rich terminal UI** — A full-featured interactive menu built with the `rich` library.
