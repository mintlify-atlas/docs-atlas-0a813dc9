## Get started

<Steps>
  <Step title="Install the bot">
    Clone the repository and install dependencies with `pip install -r requirements.txt` or use the built-in installer from the terminal menu.
  </Step>
  <Step title="Add your API keys">
    Edit `config.json` with your exchange API key, secret, and preferred settings — or use the **Settings** option in the interactive menu.
  </Step>
  <Step title="Connect a signal source">
    Choose a signal source — Telegram channel, TradingView webhook, or custom API — and configure it via the **Connect Signal Source** menu option.
  </Step>
  <Step title="Start trading">
    Run `python main.py`, select **Start Auto Trading**, and the bot will begin monitoring signals and executing trades based on your configured strategies.
  </Step>
</Steps>

<Warning>
  Trading cryptocurrency futures involves substantial risk of loss. Always start with testnet or low leverage, and never risk more than you can afford to lose. See the [Disclaimer](/reference/disclaimer) for full details.
</Warning>
